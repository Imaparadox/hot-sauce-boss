#Idea:

MVP:
User can make an account.
User can CRUD items in their shop.
User can buy an item.
User can rate an item.

Data: 
  User:
    1. Account Name
    2. Password
    3. Shop items
  Shop Item: 
    1. Picture
    2. Price
    3. Name
    4. Hot/Scoville Rating
    5. Description 
  
Search by hotness (Via tabs, no search bar)
